Página sobre

  • Este site tem o objetivo de funcionar como um meio de vendas/ loja virtual de artigos religiosos e produtos diversos,
    facilitando o processo de compra, pagamentos e envio dos produtos. Dessa forma o cliente não precisa se locomover para garantir o seu produto com qualidade e segurança.

    do IFPR Campus Paranavai, Carla Santos Oliveira e Natani G. Boço Cavalcante.
  • O desenvolvimento deste site é parte do processo avaliativo das Disciplinas de Programação para Web e Projeto de Interfaces para Web.
